# The management tool "stack" 

use [stack](https://docs.haskellstack.org/en/stable/README/) instead of `cabal`

- `stack update`
- `stack setup`: install `ghc`
- `stack ghci`, `stack ghc`, `stack runghc`, `stack exec`: new repl
- `stack install <package>`
